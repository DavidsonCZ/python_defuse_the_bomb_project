# python_defuse_the_bomb_project

Simple Python game where the player must cut the correct cables and enter the right number sequence before time runs out.

## Gameplay
Random cable color sequence (red, green, blue)
Random number sequence
Time-limited challenge
Win/lose system with sound effects

## How to run
pip install pygame customtkinter Pillow
python python_defuse_the_bomb_game.py

## Technologies used
Python
tkinter / customtkinter
pygame
Pillow

Images created by me (Adobe Illustrator)
Sound effects created by the author, with possible use of generated or library-based samples
